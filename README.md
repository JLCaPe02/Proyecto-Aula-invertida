## Índice del Material

Este repositorio contiene todo el material necesario para la implementación del aula invertida:

1.  **Material Audiovisual**: Recursos de vídeo para la fase previa.
2.  **[Documentación MkDocs](./material_aula_invertida/recurso%20Mkdocs)**: Manual completo teórico-práctico.
    -   *Disponible online en:* [https://JLCaPe02.github.io/Proyecto-Aula-invertida/](https://JLCaPe02.github.io/Proyecto-Aula-invertida/)
3.  **[Kahoot de Evaluación](./kahoot)**: Cuestionario de 50 preguntas para autoevaluación y competición en clase.
4.  **[Presentación PowerPoint](./presentacion)**: Soporte visual para la sesión presencial.
5.  **[Actividades Prácticas](./actividades)**: Guiones de prácticas para realizar en el aula.
6.  **[Examen Test](./examen)**: Banco de preguntas para la evaluación sumativa.
7.  **[Guía del Profesor](./guia_profesor/guia_profesor.md)**: Documento con instrucciones pedagógicas, tiempos y consejos para impartir la unidad.

## Reflexión Final del Grupo

La metodología de aula invertida ha permitido transformar la dinámica tradicional de la clase, moviendo la carga teórica "pesada" al trabajo autónomo del alumno y liberando el tiempo presencial para lo que realmente aporta valor: la práctica, el debate y la resolución de problemas reales.

Hemos observado que:
*   El alumnado llega a clase con dudas más concretas y profundas.
*   El tiempo de práctica efectiva se ha duplicado respecto al modelo tradicional.
*   La gamificación con Kahoot motiva la revisión de conceptos teóricos.
*   La disponibilidad de la documentación online permite consultar dudas técnicas en cualquier momento, fomentando la autonomía.

Este proyecto ha demostrado que, con una buena estructuración de materiales y una guía clara, es posible abordar temas complejos como la administración de servidores de correo de una manera dinámica y efectiva.
